# Frontend AI Handbook - 中文版

> 一本由实践驱动的指南，旨在探索和记录人工智能如何深度赋能现代前端工程。

## 🧠 缘起
33+岁，身处技术洪流。我坚信，AI不是前端工程师的替代者，而是认知外骨骼与效率杠杆。本手册是我系统性学习、实践与思考的公开记录，旨在与同行者共同绘制这个新时代的导航图。
